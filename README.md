# 🇮🇳 Hinglish Sentiment Analyzer

An interactive Machine Learning pipeline designed to analyze and classify the sentiment of mixed Hindi-English (Hinglish) text, commonly used on Indian social media platforms.

## 🚀 Overview
This project uses Natural Language Processing (NLP) and a Naive Bayes classifier to determine whether a given Hinglish sentence is **Positive**, **Negative**, or **Neutral**. 

## ⚙️ Tech Stack
* **Language:** Python
* **Machine Learning:** scikit-learn (Multinomial Naive Bayes)
* **Data Processing:** Pandas, TfidfVectorizer
